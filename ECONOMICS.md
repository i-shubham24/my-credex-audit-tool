- **Hosting (Vercel):** $0 (Hobby Tier) / $20/mo (Pro Tier for edge functions).

**Total Cost per Lead:** < $0.01

- **Average Identified Savings:** $1,200/year per startup audit.
- **Credex Service Fee:** (Assuming 10% of savings) = $120/year per lead.
- **ROI:** For every $1 spent on AI API tokens, the tool generates ~$60,000 in potential service revenue.
