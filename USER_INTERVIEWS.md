### Interview #1: Technical API vs. Seat Licensing
**Participant:** Nitin
**Role:** React Native Developer @ Sensation Software Solutions, Mohali
**Context:** Mobile agency environment with fluctuating client projects.

**Q: In your work at Sensation, how often are you integrating AI APIs into React Native apps?**
**A:** Almost every new project now. Clients want "AI-powered" everything. We usually start with a basic API call, but as the user base grows, the token costs start hitting the agency or the client hard.

**Q: Does your team track exactly which developers are using which tier of AI licenses?**
**A:** Honestly, it’s a bit of a mess. Some use GitHub Copilot, some use ChatGPT Plus personally, and we have a shared team API key. There’s definitely overlap and redundant billing.

**Q: If a tool could scan your license usage and suggest switching from seat-based licenses to raw API-direct usage, would that be useful?**
**A:** Definitely. For a React Native dev, moving logic to the backend and using a direct API instead of paying $20/month per seat for a GUI tool could save our clients a lot of overhead.

**Key Insight:** Developers are aware of the waste but lack a "receipt" to show management. The "API-direct" recommendation is the most valuable part of the audit for technical agencies.




### Interview #2: Corporate Licensing & Onboarding
**Participant:** Samridhi
**Role:** Software Engineer @ Kocher, Amritsar
**Context:** Recent hire at a larger corporate software firm.

**Q: Now that you’ve started at Kocher, what was the onboarding process like regarding AI tools?**
**A:** It was pretty standard, but I noticed everyone gets a standard "Enterprise" suite. Half the team doesn't even use the advanced AI features, but the company is paying for the full license regardless.

**Q: Do you think management knows they are paying for unused features?**
**A:** Probably not. They just see it as a "per head" cost of doing business. There's no one really auditing if a Junior Dev needs the same $30/month license as a Senior Architect.

**Q: I’m building a tool that generates a PDF audit showing exactly where to downgrade those licenses. Would your HR or Finance department actually look at it?**
**A:** If it shows them they can save 20-30% on their monthly tech bill just by clicking "Downgrade," they’d jump on it. Efficiency is a big focus right now for our local leadership.

**Key Insight:** Corporate environments suffer from "license bloat" where high-tier licenses are the default. A "Downgrade Guide" is a major selling point for Finance departments.
